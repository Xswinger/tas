### Ход работы
#### Предварительные условия
1. NVDA запущен и настроен
2. Запущен только NVDA, других активных окон и приложений нет

#### Основная последовательность шагов
1. Нажимаю кнопку Win для активации окна поиска
2. Ввожу в поле Word нажимаю Enter
3. С помощью стрелок выбираю новый документ и нажимаю Enter
4. Нажимаю Ctrl+Shift+F для выбора окна выбора шрифта, затем ввожу times new roman и нажимаю Enter. Нажимаю Ctrl+Shift+P для выбора окна выбора размера шрифта, ввожу 14 и нажимаю Enter. Для выбора межстрочного интервала нажимаю Ctrl+A для выделения всего документа и Ctrl+5 для установления интервала
5. Ввожу Титульный лист, выравниваю по центру через Ctrl+E, ввожу через Enter информацию на титульном листе. Перехожу на следующую страницу с разрывом через Ctrl+Enter. Для создания оглавления нажимаю Alt, затем S для выбора строки поиска и ввожу оглавление, с помощью стрелок перехожу в подменю предпочтительных действий и нажимаю Enter. Далее нажимаю Enter на пункте Автособираемое оглавление, делаю новый разрыв страницы через Ctrl+Enter
6. Аналогичным образом создаю заголовок введения, для применения на нем стиля заголовка выделяю его, перехожу в строку поиска через Alt и S, ввожу стили, с помощью стрелок перехожу на кнопку стилей и через Enter перехожу в окно выбора стилей, затем с помощью стрелок выбираю заголовок 1 и выхожу из окна с помощью Esc, для возвращения выравнивания по ширине при выделенном заголовке нажимаю Ctrl+E. При переходе на новую строку отключаю выранивание по ширине с помощью Ctrl+J и ввожу текст главы, при вводе глагола для создания курсива выделяю его и нажимаю Ctrl+I и повторяю со всеми встречающимися глаголами
7. Повторяю последовательность действий для следующей главы 1, за исключением глаголов: для полужирного шрифта выделяю глагол и нажимаю Ctrl+B, повторяю с каждым глаголом. Повторяю последовательность действий для следующей главы 2, за исключением глаголов: для красного цвета выделяю глагол и нажимаю Alt и S для выбора строки поиска, ввожу цвет шрифта, с помощью стрелок перехожу в подменю предпочтительных действий и нажимаю Enter. Далее в подменю выбора цвета с помощью стрелок перехожу в меню стандартных цветов и на красном цвете нажимаю Enter, возвращаюсь из строки поиска с помощью Esc, повторяю с каждым глаголом. Повторяю последовательность действий для заключения, кроме глаголов: для зеленого цвета в меню стандартных цветов выбираю зеленый цвет
8. Возвращаюсь в начало документа через Ctrl+Home, нахожу элемент оглавления, нажимаю Alt и R для перехожу в раздел ссылки, с помощью стрелок нахожу кнопку обновить таблицу и нажимаю Enter.
9. После окончания редактирования документа нажимаю комбинацию Ctrl+S, в открывшемся меню с помощью стрелок перехожу в меню сохранить, выбираю этот компьютер, нажимаю Enter и выбираю документы, нажимаю Enter, ввожу имя файла P4150_Аллаяров.word, через Tab нахожу кнопку сохранить и нажимаю Enter
10. Нажимаю Enter, сохраняя файл
11. С помощью CapsLock и стрелок нахожу кнопку закрыть окно и нажимаю CapsLock+Enter

#### Проверка наличия файла
1. Открываем проводник через комбинацию Win+E, затем используя Tab ищу дерево и используя стрелки нахожу документы и нажимаю Enter 
2. Нажимаю Tab и перемещаюсь в список файлов рабочего стола и нахожу файл, используя стрелки
3. Открываю файл нажатием Enter и проверяю его содержимое, затем с помощью CapsLock и стрелок нахожу кнопку закрыть окно и нажимаю CapsLock+Enter

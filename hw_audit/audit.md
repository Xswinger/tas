## Ozon (Версия 18.43.0)

# Тест-кейсы

**Предусловие**

Открыта начальная страница приложения

### Добавление товара в корзину

**Шаги**

1. Найти и вписать в поле поиска товара искомый товар, нажать кнопку поиска
2. Выбрать товар и перейти в его карточку
3. Изучить основную информацию о товаре (название, цена, основные характеристики)
4. Найти и активировать кнопку "В корзину"

**Ожидаемый результат**

- Пользователь не сталкивается с трудностями при прохождении сценария
- Все элементы доступны для пользователя через скринридер

**Фактический результат**

- Сценарий пройден
- Отдельные элементы карточки товара имеют трудноразбираемые названия (иконки отзывов и оценки имеют неразбираемый альтернативный текст), кнопки фильтров недоступны пользователю через скринридер, значительная часть вспомогательных элементов на странице товара (добавление в избранное, поделиться ссылкой, вернуться в список товаров) недоступна для скринридера

### Просмотр купленного товара

**Шаги**

1. Перейти в профиль через поиск и двойной тап кнопки меню профиля
2. Перейти на страницу покупок через поиск и двойной тап кнопки "Покупки"
3. Перейти в меню заказов через поиск и двойной тап кнопки "Заказы"
4. Перейти в подраздел завершенных заказов через двойной тап кнопки "Завершенные"
5. Найти панель с заказанным товаром, перейти в эту панель и найти целевой товар, перейти в его карточку

**Ожидаемый результат**

- Пользователь не сталкивается с трудностями при прохождении сценария
- Все элементы доступны для пользователя через скринридер

**Фактический результат**

- Пользователь не может перейти в профиль по причине нечитаемости кнопок нижнего меню с помощью скринридера
- Сценарий не пройден

### Оценка купленного товара

**Шаги**

1. Перейти в профиль через поиск и двойной тап кнопки меню профиля
2. Перейти на страницу оценки через поиск и двойной тап кнопки "Ждут отзыва"
3. Выбрать среди списка товар, который нужно оценить
4. Выставить оценку в количестве звезд с помощью специального поля, написать текст в поле ввода текста, прикрепить фотографии и активировать кнопку отправить отзыв

**Ожидаемый результат**

- Пользователь не сталкивается с трудностями при прохождении сценария
- Все элементы доступны для пользователя через скринридер

**Фактический результат**

- Пользователь не может перейти в профиль по причине нечитаемости кнопок нижнего меню с помощью скринридера
- Сценарий не пройден

### Другие проблемы

- Некоторые всплывающие окна (например оценки приложения Ozon) невозможно закрыть при причине недоступности кнопки закрытия для скринридера
- Большая часть интерактивных элементов на главной странице недоступны для скринридера или не имеют альтернативного описания (кнопки других сервисов Ozon, фотографии товаров и т.п.)

## Яндекс GO (Версия 700.119.0)

# Тест-кейсы

**Предусловие**

Открыта начальная страница приложения

### Добавление нового адреса

**Шаги**

1. Перейти в меню профиля через поиск и двойной тап кнопки меню
2. Перейти в меню моих адресов через поиск и двойной тап кнопки "Мои адреса"
3. Добавить новый адрес, выбрать тип адреса через активацию соответствующих кнопок
4. Ввести в поле новый адрес дома, дать ему название, сохранить

**Ожидаемый результат**

- Пользователь не сталкивается с трудностями при прохождении сценария
- При успешном сохранении адреса пользователь будет возвращен в меню профиля или добавления нового адреса
- Все элементы доступны для пользователя через скринридер

**Фактический результат**

- Сценарий пройден
- При успешном сохранении адреса пользователь остается в меню ввода дополнительной информации о новой точке, скринридер при этом успешном сохранении озвучивает "Сохранение", а затем "Недоступно" 
- Все элементы доступны через скринридер

### Повторение заказа еды

**Шаги**

1. Перейти в меню профиля через поиск и двойной тап кнопки меню
2. Перейти в меню моих заказов через поиск и двойной тап кнопки "История заказов"
3. Найти панель последнего заказа, активировать кнопку "Повторить"
4. Проверить позиции в заказе, нажать кнопку "К оплате"

**Ожидаемый результат**

- Пользователь не сталкивается с трудностями при прохождении сценария
- Все элементы доступны для пользователя через скринридер

**Фактический результат**

- Сценарий успешно пройден
- Все элементы доступны через скринридер, некоторые элементы (например компонент выбора количества штук в позиции заказа) имеет расширенное описание

### Добавление товара в корзину в Маркете

**Шаги**

1. Найти и активировать кнопку яндекс маркета в горизонтальном скролле сервисов
2. Найти и вписать в поле поиска товара искомый товар
3. Перейти в карточку товара, изучить основную информацию о нем (название, цену, основные характеристики)
4. Найти и активировать кнопку "В корзину"

**Ожидаемый результат**

- Пользователь не сталкивается с трудностями при прохождении сценария
- Все элементы доступны для пользователя через скринридер

**Фактический результат**

- Сценарий пройден
- При работе с горизонтальным скроллом пользователю не совсем понятно, что это скролл (нет подсказки о возможности пролистнуть списов влево), нет подсказки об окончании загрузки содержимого при перехоже в сервис маркета
- Из меню просмотра карточек для скринридера не доступна информация о сроках доставки (хотя она присутствует на карточке)

### Другие проблемы

- Нижняя панель сервиса Лавка в отдельных случаях может быть труднодоступна из-за ее наслаивания на карточки продуктов, в результате чего скринридером обнаруживаются элементы карточек товаров, а не нижней панели
